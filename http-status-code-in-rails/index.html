    HTTP Status Code In Rails

    Outline

    • 使用方式
    • 這些Symbol其實定義在Rack中

    Rails將這些HTTP Status定義成有意義的單字。讓我們可以在使用的時候減少因為背錯而發生錯誤的機率。

    使用方式

    用symbol加上單字即可取代原本的HTTP Status Code(500)

    render status: 500
render status: :forbidden

    這些Symbol其實定義在Rack中
