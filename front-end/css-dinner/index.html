    CSS Dinner

    CSS Diner 是一個很有趣的小遊戲，你必須使用css來選取桌上扭動的物體， 全部26關玩完對 CSS selector 想忘也忘不了。 css selector 在 javascript 或是 jQeury 也會用到，不熟的話到時候還是需要花額外的時間加強。 有神人整理成一系列好玩練習效果又顯著的教學，好教學不玩嘛？XD

    學習目標

    1. Univeral, Type selector, Class selector, ID selector的用法。
    2. 了解ul > ui {}和ul ui{}的不同
    3. 了解A+B與A~B的不同。
    4. 了解pasudo selector的用法。

    Memo

    1. A+B 與 A~B 的不同

    A+B 代表與 A 相鄰的 B。A~B 代表與 A 同一層的 B 都選。
